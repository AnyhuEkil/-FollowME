05.10 강사님께 프로젝트 진행사항 보고할 내용.

■ 이번 프로젝트 컨셉 : 프로젝트 로직 이해와 그 동안 프로젝트 문제점 파악 및 보완

■ 지난 문제점
 완성물을 목표로 프로젝트 진행
 과한 욕심 대비 부족한 실력
 -> 결과물은 창출했으나, 과정에서 몇 가지 문제가 발생.
 (프로젝트 전체 진행 과정은 전 조장(박윤하)가 리더쉽으로 충분히 잘 이끌었음. 반면,  완성물보다는 프로젝트 진행 전반적인 이해 및 타인의 코드에 대한 이해가 부족했음.) 

■ 예상되는 문제점 :
 현 조장(서종원)이 JAVA 및 JSP에 낮은 성취상태로, 프로젝트 전체적인 그림을 그리기 힘듬.
 현 조장(서종원)이 프로젝트 진행에 대한 이해도가 낮아, 매끄러운 진행은 어려워보임.

■ 문제 해결 방안
 프로젝트 크기 축소
  -> 설계 및 기획 기간 연장 및 구현기간 축소
  -> 협업 중시
 프로젝트 진행 과정 습득
 -> 강사님께서 해주시는 피드백 적극활용.
 -> 조원의 강한 피드백 적극 수용(진행 과정에 한함)
 -> 
 -> 협업중시

■ 예상되는 결과물
 JAVA 와 JSP를 활용한 어플리케이션(최소한의 HTML, CSS 및 부가사항)
 조원의 협업 능력 향상 


1주차 : JAVA 및 개인 공부에 치중 + 주제선정이 목표 (완료)
2주차: 명료화된 계획 -> 일정 조정, 스토리 플로우, 요구사항 정의, 데이터베이스 명세화 등
3주차 : 업무 분장, 기능 구현 및 디자인
4주차 : 기능 구현, 테스트 및 코드 리뷰
5주차 : 서류작업 및 마무리
